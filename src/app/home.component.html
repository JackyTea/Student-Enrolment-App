Web Service

A Heroku hosted Node Express web API designed to fetch student data via Mongoose.js library methods. All of the data is stored in a MongoDB cloud cluster and is retrieved in JSON format for ease of parsing. There are methods to pass GET, POST, PUT and DELETE requests to the database for user manipulation of the data.

Web Service

Web Application

A Heroku hosted Angular web application designed to render and manipulate fetch API data through a user-friendly interface. Applies the concept of HTML form patterns through CRUD (create, read, update, delete) handling of database data.
