Student Enrolment Angular Application

This application implements the following:

HTML Form Patterns

  • Get All - (List, GET Method)
  • Get All - Restricted (Filtered List, GET Method w/ Mongoose Tasks)
  • Get One - (Detail, GET Method w/ Unique Identification Value)
  • Edit (PUT Method)
  • Create (POST Method)
  • Delete (DELETE Method)

Web Service API Fetch Interactions

  • Data is fetched from a remote service in JSON format to be rendered in the app.

Angular Routing

  • Routing for page navigation, redirection and 404 routes are implemented for ease of traversal.

Angular Fundamentals

  • Fundamental Angular concepts used here would include directives, interfaces, services, etc...
